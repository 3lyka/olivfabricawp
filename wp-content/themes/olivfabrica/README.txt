Тут будет душный текст, как работать с этими файлами

index.html собирается из папки block-item, там есть куски и их удобно редактировать
catalog хранятся папки с файлами и выводами, переходят в карточку товара
css как css
img - это картинки index.html 
main.parts в ней находится head header footer, они идут вставкой на все страницы, можно поредачить и вывод будет намного удобнее
page - временная папка для расперделения файлов

Это штука для работы с категориями товаров, душно, но терпимо, через jQuery настроена фильтрация, тянется с базы SQL, далее будет понятно

Первое правило работы с базой данных НЕ СТАВЬ "-", по каким то причинам крашит


id- поле не заполняется (оно нужно для обработки)
tittle - название товара
style - стиль кухни (классический, хай-тек, скандинавский)
material - материал из чего сделан товр (дсп, камень и т.п.)
img - название изображения
category - тоже самое что и style только здесь нужно написать англискими буквами (hi-tech, classic-style) , нужно для того чтобы присвоить фильтрацию товара
price - цена
old_price - старая цена (может не быть)
sale- скидка (может не быть)
